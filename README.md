# README

## **IsDefinition** - is a simple app that makes it easy to memorize your definitions.

#### Installation guide *(npm required!)*

>Clone git repository:
>
>- `git clone https://github.com/l3r8yJ/isDefinitionApp.git`
>
>Chage directory:
>
>- `cd isDefinitionApp`
>
>Run make: 
>
>- `npm run make`
